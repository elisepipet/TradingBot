# 📈 TradingBot

Automated trading algorithm in Python. Analyzes market data and executes orders based on customizable strategies.

## Tech Stack
- **Language** — Python, Shell
- **Data** — CSV market data

## Features
- Automated market analysis from CSV data
- Order execution based on trading strategies
- Customizable strategy parameters
- Bonus strategies included
- Test suite for strategy validation

## Getting Started

### Prerequisites
- Python 3.x

### Installation
```bash
pip install -r requirements.txt
```

### Usage
```bash
./trade
```

See `exemple` for usage examples.

## Project Structure
- `/csv` — Market data files
- `/tests` — Test suite
- `/bonus` — Additional strategies
